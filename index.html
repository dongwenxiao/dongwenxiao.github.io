sp-boilerplate (super-project 模板项目)

目录 /apps 说明

apps下承载一下子项目，每个子项目可能有自己的前端和后端。

  • api 接口服务模板
  • doc React同构模板 & 文档站点
  • www EJS网站模板
  • saas 第三方服务功能模板

Debug

使用debug模块做控制台输出，类型：info|error|debug|log
